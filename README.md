# Gross Margin Calculator

A CQuence Health tool for calculating gross margins, break-even points, and optimizing pricing with fixed cost analysis.

## Features

### Section 1: Unit Economics
- Revenue per unit input
- COGS (Cost of Goods Sold) per unit
- Projected units
- Real-time gross margin calculation (% and $)

### Section 2: Fixed Costs & Pricing Analysis (Collapsible)
- Pre-populated fixed cost categories (Rent, Salaries, Insurance, Utilities, Marketing)
- Add/remove custom cost categories
- Total fixed costs calculation
- Target margin pricing (reverse-engineer price from desired margin)
- Break-even units calculation
- Net profit and net margin analysis
- Contextual pricing insights

### Additional Features
- Industry margin benchmarks
- Usage tips and guidance
- PDF export
- Copy results to clipboard

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Lucide React (icons)
- jsPDF (PDF generation)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the calculator.

## Deployment

This app is configured for Azure Static Web Apps deployment. See `.github/workflows/` for CI/CD configuration.
